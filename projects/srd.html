For this project, we designed a Fully Normalized Relational Database to support the operations of a fictitious flower shop based in Portugal. The goal was to store and retrieve business-critical data including sales, stock, customer interactions, product ratings, and employee performance.

Using MySQL Workbench, we created an ERD with 8+ entities and implemented the system using SQL scripts. We also created triggers to automatically update stock levels and log changes, inserted mock transaction data over a two-year period, and answered key business questions through advanced SQL queries. To demonstrate business reporting capabilities, we generated invoices using dynamic views in MySQL.

The final solution is fully functional and demonstrates both my technical ability and business-oriented thinking in designing structured, scalable databases.
