Financial Sentiment Analysis with NLP

Python (e.g., Scikit-Learn, nltk, transformers)

In this project, we built a sentiment classification system that detects whether financial tweets are bearish, bullish, or neutral.

We tested a range of models, from traditional machine learning (Logistic Regression, Random Forest) to advanced transformer models (DistilBERT, FinBERT, GPT-2). DistilBERT delivered the best results, achieving 87% accuracy.

The project also includes custom preprocessing, named entity recognition, and emoji analysis. This work highlights the power of transformers for financial NLP and the importance of adapting modeling to domain-specific text.
